# Де світло? ІФ

Простий маленький проєкт для перевірки погодинних графіків вимкнень електроенергії в Івано‑Франківську.

## Опис

- `GET /search_by_address?address={address}` - шукає чергу за адресою.
- `GET /search_by_queue?queue={queue}` - повертає розклад за номером черги.

## Встановлення

```bash
npm install
```

## Запуск

```bash
node index.js

npm start
```

Відкрийте http://localhost:3000

## Використання
- Введіть адресу у формі або виберіть чергу і натисніть кнопку - фронтенд зробить запит до локального сервера.
